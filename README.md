# 🗡️ 杀戮尖塔 2 - 十六夜咲夜 皮肤 Mod (Sakuya Izayoi Skin)

> “你的时间属于我了。—— 完美潇洒的从者降临高塔！” 

本项目是为《杀戮尖塔 2》(Slay the Spire 2) 制作的皮肤替换 Mod。它将游戏中的原版角色 **静默猎手 (The Silent)** 极其华丽地全方位替换为《东方 Project》中红魔馆的完美女仆长 —— **十六夜 咲夜 (Sakuya Izayoi)**！

## ✨ 核心特性 (Features)

本项目并非简单的贴图替换，而是从底层进行了极其硬核的逻辑劫持与视听重构，包含以下特性：

* 🔪 **极其致命的“小刀流”专属适配**：
  * 为猎手的核心机制“小刀 (Shiv)”量身定制了专属的投掷动画（多段随机）与极其干脆短促的专属配音（如「行きなさい」），一回合狂扔十几把飞刀的沉浸感直接爆表！
* 🎬 **华丽收场 (Grand Finale) 专属处决彩蛋**：
  * 赛博雷达极其精准地拦截底层的“华丽收场”指令！打出这张终结技时，完美女仆不再进行普通攻击，而是极其优雅地行一个淑女礼作为终极处决动作！
* 😭 **电影级多段式战损反馈系统 (Death UI Overhaul)**：
  * 极其霸气地劫持了原本冷冰冰的 Game Over 墓碑界面。战败时，将触发长达数秒的 3 段式凄美淡入淡出演出，并随机抽取展示多张战损版表情差分，把挫败感化作极其强烈的保护欲！
* 🎇 **M.U.G.E.N 级双轨特效引擎**：
  * 重构了底层特效系统，打出技能牌时将极其华丽地引爆 2D 逐帧光影特效（支持发光叠加混合模式），采用五连抽盲盒系统，彻底告别光污染。
* 📢 **沉浸式动态战术语音库 (Voice Lines)**：
  * 收录了战斗入场、受击、专属防御支援、以及精英战/Boss战专属斩杀结算台词，全程极其高冷且优雅。
* ⚙️ **赛博镜像雷达同步**：
  * 完美继承 V1 架构的底层黑科技，内置高频监听雷达，实现极其丝滑的被夹击前后转身同步。

## 📦 安装指南 (Installation)

1. 确保你已经安装了《杀戮尖塔 2》(Slay the Spire 2)。
2. 在 GitHub 的 [Releases 页面] 下载最新的 Mod 压缩包。
3. 将解压后得到的文件夹（包含 `.dll`、`.json` 以及 `.pck` 文件）放入你的游戏 Mod 根目录：
   `你的游戏安装路径/Slay the Spire 2/mods/`
4. 启动游戏，在游戏内的 Mod 管理器中极其优雅地勾选本 Mod 即可生效！

## 🛠️ 极客架构说明 (For Developers)

本项目的源码极其清爽，采用现代 C# 与 Godot 4.x 编写，并利用 Harmony 库进行底层劫持。
为了保证极客级别的美学与执行效率，项目采用了极其严谨的模块化架构：
* `SakuyaGlobals.cs`: 赛博数据中枢（包含所有的状态缓存、弹药库路径常量、随机数引擎，以及按需懒加载协议）。
* `Patch_Combat.cs`: 战斗全生命周期劫持（包含“华丽收场”卡牌精准拦截、M.U.G.E.N 特效节点扫描与引爆、动作语音调度）。
* `Patch_DefeatView.cs`: 极其暴力的战败结算界面 UI 注入与 `Tween` 补间动画演出引擎。

如果你也想为《杀戮尖塔 2》开发底层逻辑 Mod，本仓库的代码极其欢迎你随意白嫖、参考与魔改！
